# TLM_Installer
One click installer of TLM-webui for Windows

Needs:
- Python 3.10.6
- Git
- This exe latest version

How to:
1. Install Python 3.10.6 and Git
2. Install exe installer latest version
3. Start exe installer
4. After cloning delete exe installer
5. Run ```start.bat```
